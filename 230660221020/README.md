## Deskripsi Repository

Repository ini berisi kumpulan *praktikum mata kuliah Pemrograman Berbasis Web (PBW)*  
yang dikerjakan secara bertahap mulai dari *praktikum 1 hingga praktikum 16*,  
sesuai dengan modul dan materi perkuliahan yang diberikan oleh dosen pengampu.

Setiap folder merepresentasikan *satu topik praktikum*, yang mencakup konsep dasar
hingga lanjutan dalam pengembangan web menggunakan JavaScript dan teknologi backend.

---

## Struktur Praktikum

Berikut adalah daftar praktikum yang terdapat dalam repository ini:

- **01 – Dasar NodeJS dan JavaScript**  
  Pengenalan Node.js serta konsep dasar JavaScript.

- **02 – Function dan Conditional**  
  Pembahasan fungsi, percabangan, dan logika dasar program.

- **03 – Looping dan Array**  
  Penggunaan perulangan dan pengolahan data array.

- **04 – Object Literal**  
  Implementasi object literal dalam JavaScript.

- **05 – JavaScript Modern (ES6+)**  
  Fitur modern JavaScript seperti arrow function, destructuring, dan module.

- **06 – Object Oriented Programming (OOP) di JavaScript**  
  Konsep class, inheritance, encapsulation, dan abstraction.

- **07 – Asynchronous JavaScript**  
  Callback, Promise, dan Async/Await.

- **08 – Database**  
  Konsep dasar database dan integrasi dengan aplikasi.

- **09 – TypeScript Fundamental**  
  Pengenalan TypeScript dan penerapannya dalam pengembangan aplikasi.

- **10–16 – Express.js**  
  Implementasi backend menggunakan Express.js, termasuk routing, middleware,
  dan pengembangan REST API sederhana.

- **UTS**  
  Hasil pengerjaan Ujian Tengah Semester mata kuliah PBW.
